# @nic/mfe-lib-shared 가이드

**`@nic/mfe-lib-shared`** 는 멀티레포 마이크로 프론트엔드 아키텍처에서
모든 리모트 앱이 공통으로 사용하는 공유 라이브러리입니다.

이 Storybook은 공유 라이브러리의 모든 기능에 대한 **인터랙티브 문서** 및 **사용 가이드**를 제공합니다.

---

## 라이브러리 구성

### UI Components

[shadcn/ui](https://ui.shadcn.com/)를 기반으로 커스터마이징된 공통 UI 컴포넌트들입니다.
Tailwind CSS와 Base UI 위에서 구축되어 접근성과 디자인 일관성을 보장합니다.

| 컴포넌트  | 설명                                       |
| --------- | ------------------------------------------ |
| `Button`  | 다양한 variant와 size를 지원하는 기본 버튼 |

```tsx
import { Button } from "@nic/mfe-lib-shared/components/ui";
// 또는
import { Button } from "@nic/mfe-lib-shared/components";
```

---

### Hooks

API 호출 및 서버 상태 관리에 활용할 수 있는 TanStack Query 기반의 커스텀 훅들입니다.

```ts
import { useApi } from "@nic/mfe-lib-shared/hooks";
```

---

### Utils Function

#### API Utils

`callApi` — Axios 기반의 통합 HTTP 요청 함수입니다.
인증 헤더 주입, 에러 핸들링, 응답 타입 추론을 자동으로 처리합니다.

```ts
import { callApi } from "@nic/mfe-lib-shared/utils";

const { data } = await callApi<User[]>("/api/users");
```

---

### Types

앱 전반에서 공유하는 TypeScript 타입 정의들입니다.

```ts
import type { ApiResponse } from "@nic/mfe-lib-shared/types";
```

| 모듈           | 내용                     |
| -------------- | ------------------------ |
| `types`        | API 응답, 공통 객체 타입 |
| `types/router` | 앱 라우터 관련 타입      |

---

### 공통 설정 (Config)

각 리모트 앱에서 재사용할 수 있는 도구 설정 프리셋입니다.

```ts
// eslint.config.js
import reactConfig from "@nic/mfe-lib-shared/config/eslint";
export default [...reactConfig];

// prettier.config.mjs
import prettierConfig from "@nic/mfe-lib-shared/config/prettier";
export default prettierConfig;
```

---

### Common Styles

CSS 변수(Design Tokens)와 전역 스타일이 정의되어 있습니다.
각 앱 진입점에서 한 번만 임포트하면 전체 테마가 적용됩니다.

```ts
import "@nic/mfe-lib-shared/styles";
```

## 이 Storybook 활용하기

왼쪽 사이드바에서 카테고리를 선택하세요.

- **Getting Started** — 소개 및 설치 가이드 (현재 페이지)
- **Components** — UI 컴포넌트 인터랙티브 데모 및 Props 문서
- **Functions/Hooks** — 커스텀 훅 사용법
- **Functions/Utils** — 유틸리티 함수 사용법

각 컴포넌트 페이지에서는 다음을 활용할 수 있습니다.

| 탭                | 설명                                            |
| ----------------- | ----------------------------------------------- |
| **Docs**          | 컴포넌트 개요, Props 테이블, 코드 예시          |
| **Canvas**        | 컴포넌트 실시간 렌더링 및 Controls로 Props 조작 |
| **Accessibility** | 접근성(a11y) 진단 결과                          |

## 설치 및 임포트

공유 라이브러리는 멀티레포 내부 패키지이므로 npm 워크스페이스로 자동 연결됩니다.

```json
{
  "dependencies": {
    "@nic/mfe-lib-shared": "*"
  }
}
```

```tsx
// 컴포넌트
import { Button } from "@nic/mfe-lib-shared/components/ui";

// 유틸리티
import { callApi } from "@nic/mfe-lib-shared/utils";

// 타입
import type { ApiResponse } from "@nic/mfe-lib-shared/types";

// 스타일 (앱 진입점에서 1회)
import "@nic/mfe-lib-shared/styles";
```
